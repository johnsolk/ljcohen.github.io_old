1. Experiment Description

This is a differential gene expression analysis from eleven custom Agilent microarray chips processed at HBOI on six dates: 12/5/2013, 5/24/2013, 5/30/2013, 6/11/2013, 6/5/2013, 6/6/2013. The experiment consists of hard coral samples, Montastraea cavernosa collected from 3 sites on 3 different dates. At least 4 samples at each site were collected from separate colonies for a total of 44 samples.

2. Data analysis procedure

Microarray chips were imaged at 635 nm using a high-resolution laser GenePixÂ® 4200A microarray scanner (Axon, Molecular Devices, California, USA). Data were extracted from images using GenePixPro 6.0 software as .gpr files.

Data were imported into the R statistical programming environment. The R-Bioconductor package, limma (Linear Models for Microarray Data, version 3.22.3, Ritchie et al. 2015; Smyth 2004) was used to assess the quality of the data and to analyze for differential expression.
